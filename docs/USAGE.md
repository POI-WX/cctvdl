# 使用指南 / Usage

本指南介绍 cctvdl 的完整使用流程。安装见 [README](../README.md)，遇到问题见 [常见问题](FAQ.md)。

总体流程：**导入节目 → 浏览/搜索 → 勾选并选清晰度 → 下载 → 在下载页管理与打开**。

## 📥 1. 导入节目

1. 在浏览器打开你想下载的央视节目页或栏目页，复制地址栏里的链接（`tv.cctv.com` 下的页面）。
2. 打开 cctvdl，在首页顶部的导入栏粘贴链接并回车；也可以直接把链接拖放到窗口。
3. 导入成功后，栏目会出现在左侧列表，已导入的栏目下次启动会自动保留。

**管理栏目**：把鼠标移到某个栏目上，右侧会出现两个按钮——

- **⭐ 收藏** — 点亮后该栏目置顶到「收藏」分组（最近收藏的在最上）；再点取消。
- **🗑 删除** — 删除该栏目（也可右键栏目，或选中后按 `Delete` 键；macOS 用 `Delete` 或 `Fn+Delete`）。

栏目较多时顶部会出现搜索框；表头的 🧹 可一键清空全部栏目（带确认）。

![首页](assets/home.png)

## 🔍 2. 浏览与搜索

- 点击左侧栏目名称，加载该栏目的视频列表。
- 用顶部搜索框按标题 / 简介 / 日期过滤；切换月份查看不同时间的更新。
- 视频卡片显示标题、简介与时间；点击封面可全屏预览，预览中还可一键复制标题或简介。

![大图预览](assets/lightbox.png)

## 🎚️ 3. 选择清晰度并下载

1. 勾选要下载的一个或多个视频（`Ctrl+A` 可全选）。
2. 在设置中选择清晰度，再点「下载选中」，任务进入下载队列。

可选清晰度档位（不同节目实际提供的档位不同，缺档时自动降级到最接近的可用档）：

| 档位 | 大致分辨率 |
|------|-----------|
| 流畅 | 270p |
| 标清 | 360p |
| 高清 | 720p |
| 超清 | 720p（更高码率） |
| 蓝光 | 1080p |
| 自动 | 取当前节目可用的最高档 |

## 📊 4. 查看进度与管理任务

下载页把任务分为三组（可折叠）：**进行中** / **已完成** / **失败·取消**。

- **进行中** — 每张卡片显示状态、速度、剩余时间与进度；可单独「取消」，或用顶部「全部取消」。
- **已完成** — 卡片提供两个操作：
  - **▶ 播放** — 用系统默认播放器打开视频。借此可立即确认下载是否成功、能否正常播放。
  - **📂 文件夹** — 在文件管理器中定位该文件。
  - 下载完成的视频以 **MP4** 保存到设置中的保存目录。
- **失败·取消** — 点「↺ 重试」即可，**重试会自动从断点续传**，已完成的片段不会重新下载；顶部「重试失败」可批量重试。

顶部工具栏还提供「清除完成」与「打开文件夹」。已下载的视频会被记录，重复下载时自动跳过（可在设置中清空历史）。

## ⚙️ 5. 设置

![设置](assets/settings.png)

| 设置项 | 说明 |
|--------|------|
| 文件保存位置 | 视频输出目录（需有写权限） |
| 并发下载数 | 同时下载的片段数，默认 8，范围 1–16；网络好可调高 |
| 视频清晰度 | 默认下载档位，缺档自动降级 |
| 合并方式 | 默认「无损快速」（画质无损、速度快）；个别片源播放异常时可切「兼容重编码」 |
| 深色模式 | 浅色/深色切换，下次启动自动恢复 |
| 主题色 | 6 种预设主题色，即时生效 |
| 日志级别 / 日志目录 | 日志详细程度（**日常** / **调试**）与存放位置；排查问题时切到「调试」会记录更多细节 |
| 下载历史 | 查看或清空已下载记录 |

## 🎨 界面图标

| 图标 | 含义 |
|:---:|------|
| 📺 | 左上角图标，点击查看「关于」与版本 |
| 🏠 / ⬇️ / ⚙️ | 首页 / 下载 / 设置 |
| ▶ | 播放（用系统播放器打开） |
| 📂 | 在文件管理器中定位文件 |
| ↺ | 重试失败任务 |
| ⏹ | 全部取消 |
| ✓ / ✗ | 已完成 / 失败 |
| 📋 / 📄 | 复制标题 / 复制简介 |

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `1` / `2` / `3` | 切换 首页 / 下载 / 设置 |
| `F5` | 刷新视频列表 |
| `Ctrl+A` | 全选 / 取消全选视频 |
| `Delete` | 删除选中的栏目（焦点不在输入框时；macOS 也可用 `Fn+Delete`） |
| `Esc` | 关闭弹窗（封面大图、关于） |

## ✨ 更多体验

- **侧边栏**可展开 / 收起（点底部箭头切换）。
- **深色模式 / 主题色**在设置中切换，下次启动自动恢复。
- **系统托盘**常驻并显示下载进度；单击托盘图标显示窗口，右键弹出菜单（Linux 个别发行版只能用菜单）。
- **窗口大小与位置**会被记住；重复启动会自动聚焦已打开的窗口。
- 支持把链接**直接拖拽**到窗口快速导入。

## ℹ️ 关于与版本

点击左上角的 📺 图标可打开「关于」弹窗，查看当前版本号——反馈问题时附上版本号会更便于定位。
