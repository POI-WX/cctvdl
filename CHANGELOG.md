# Changelog

本项目的所有重要变更记录在此文件中。

---

## 0.3.2

### 新增 / 优化

- **下载历史详情**：设置页历史记录升级为列表视图，展示标题、完成时间（相对）、文件大小、文件名；支持单条删除（🗑）与定位文件（📂）
- **下载页观感**：速度数字加粗独立行、进度条条纹动画（下载中）、排队序号（第 N 位）、批次完成右下角通知、所有卡片支持右键菜单（复制标题 / 播放 / 定位 / 重试）

---

## 0.3.1

### 新增

- **单视频集合备份**：「📌 单个视频」集合支持从 JSON 导出备份 / 从 JSON 导入恢复，与栏目备份对称；集合表头新增 ↓/↑ 操作按钮
- **下载历史富模型**：历史记录从仅存 guid 升级为含标题、文件路径、文件大小、完成时间的完整条目；自动兼容迁移旧格式
- **下载队列持久化**：应用异常退出后重启可自动恢复上次未完成的排队任务，不再需要重新手动选择下载

---

## 0.3.0

### 优化

- **代码质量与性能整体优化**：修复 App.vue 重复订阅 IPC 事件导致 store 被调用两次；下载进度回调三次 `segments.filter` 合并为单次遍历；收藏栏目新内容检查改为并发请求；类型全面收紧（`JobState`/`JobStage`/`Quality` 替换 `string`）；冗余 computed 从已有派生消除重复计算

---

## 0.2.5

### 新增 / 优化

- **Pinia 状态管理**：`useAppStore` / `useDownloadStore` / `useContentStore` 统一组件状态，IPC 事件监听集中在 store action 中
- **结构化错误码**：`errors.ts` 新增 `ErrorCode` 枚举与 `AppError` 类型，`humanizeError` 签名向后兼容
- **单视频封面与简介修复**：优先从 `getHttpVideoInfo` API 获取真实封面（fmspic 截帧）和简介，不再显示 CCTV 官网通用占位图
- **单视频 guid 修复**：优先取页面 HTML 中的 `var guid`（实际可下载 guid），而非 URL 中的 VIDE token（CMS 内容 ID）
- **僵尸栏目检测**：导入栏目时自动检测是否有历史内容，无内容则回退到单视频下载（修复停播节目与独立电影被误导入为空栏目的问题）
- **首页视频区加宽**：左侧内容面板从 300px 加宽至 360px，视频行标题不再截断

## 0.1.0

cctvdl 的首个公开版本：一款开箱即用的跨平台央视视频下载器。以下是本版本提供的能力。

### 下载

- 导入栏目 / 节目链接，按月浏览、关键词搜索，封面与简介预览
- 多选批量下载，串行队列执行，任务卡片实时显示进度、速度与剩余时间
- 自动处理央视加密节目，下载后即可正常观看
- 清晰度可选：流畅 / 标清 / 高清 / 超清 / 蓝光（最高 1080p），或「自动」取最高，缺档自动降级
- 下载完成自动合并为 MP4（默认无损快速，可选兼容重编码）
- 断点续传：失败 / 取消后保留进度，重试自动续传
- 下载历史去重，重复内容自动跳过

### 管理与界面

- 下载页分「进行中 / 已完成 / 失败·取消」三组；完成后可一键播放（系统播放器）或定位文件
- 垂直侧边栏（可展开 / 收起）、深色模式、6 种主题色、封面大图预览
- 拖放导入、键盘快捷键（`1`/`2`/`3`、`F5`、`Ctrl+A`、`Delete`、`Esc`）、系统托盘进度提示、窗口大小 / 位置记忆
- 设置：保存位置、并发数、清晰度、合并方式、深色模式、主题色、日志级别 / 目录、下载历史管理

### 开箱即用

- 支持 Windows / macOS / Linux
- 安装包已内置所需运行时与 ffmpeg，无需任何额外安装或配置

### 质量

- 三层测试：单元测试 + GUI 自动化测试 + 端到端联网管线测试
- CI 在三平台逐层把关（lint / typecheck / build → 单元 → GUI）；端到端联网测试因 CDN 封境外 IP 改为手动触发
- ESLint；打 `v*` tag 时自动构建并发布三平台安装包
