# Changelog

本项目的所有重要变更记录在此文件中。

---

## 0.2.0

### 新增

- **单视频直链下载**：粘贴不属于任何栏目的独立视频（如电影）链接，自动识别并直接下载
- 左侧「我的栏目」升级为「我的内容」，新增常驻「📌 单个视频」集合——单视频持久保存，可预览 / 下载 / 移除，与栏目浏览隐式切换

---

## 0.1.2

栏目管理增强与特殊栏目修复。

### 新增

- **收藏栏目**：⭐ 置顶常用栏目（最近收藏的排最前），列表按「收藏 / 全部」分组显示
- **批量清空栏目**：表头 🧹 一键清空（带确认）
- 栏目行悬停显示收藏 / 删除按钮，删除不再只能靠右键

### 修复

- macOS 下 `Delete` 键无法删除栏目（现支持 `Delete` 与 macOS 的 `Backspace`）
- 在搜索框中按删除键会误删选中栏目（输入框聚焦时不再触发删除快捷键）
- 「等着我」等无标准 `column_id` 的特殊栏目页现明确报错，不再导入取不到视频的空栏目

## 0.1.1

### 修复

- 更新 README / FAQ 的 macOS 安装与放行说明，补充终端 `xattr` 临时绕过方式

## 0.1.0

cctvdl 的首个公开版本：一款开箱即用的跨平台央视视频下载器。以下是本版本提供的能力。

### 下载

- 导入栏目 / 节目链接，按月浏览、关键词搜索，封面与简介预览
- 多选批量下载，串行队列执行，任务卡片实时显示进度、速度与剩余时间
- 自动处理央视加密节目，下载后即可正常观看
- 清晰度可选：流畅 / 标清 / 高清 / 超清 / 蓝光（最高 1080p），或「自动」取最高，缺档自动降级
- 下载完成自动合并为 MP4（默认无损快速，可选兼容重编码）
- 断点续传：失败 / 取消后保留进度，重试自动续传
- 下载历史去重，重复内容自动跳过

### 管理与界面

- 下载页分「进行中 / 已完成 / 失败·取消」三组；完成后可一键播放（系统播放器）或定位文件
- 垂直侧边栏（可展开 / 收起）、深色模式、6 种主题色、封面大图预览
- 拖放导入、键盘快捷键（`1`/`2`/`3`、`F5`、`Ctrl+A`、`Delete`、`Esc`）、系统托盘进度提示、窗口大小 / 位置记忆
- 设置：保存位置、并发数、清晰度、合并方式、深色模式、主题色、日志级别 / 目录、下载历史管理

### 开箱即用

- 支持 Windows / macOS / Linux
- 安装包已内置所需运行时与 ffmpeg，无需任何额外安装或配置

### 质量

- 三层测试：单元测试 + GUI 自动化测试 + 端到端联网管线测试
- CI 在三平台逐层把关（lint / typecheck / build → 单元 → GUI）；端到端联网测试因 CDN 封境外 IP 改为手动触发
- ESLint；打 `v*` tag 时自动构建并发布三平台安装包
