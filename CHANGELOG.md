# Changelog

本项目的所有重要变更记录在此文件中。

---

## 0.4.3

### 新增

- **保存封面**：预览面板操作栏（复制简介右侧）新增「🖼 保存封面」按钮，一键将视频封面图片保存到本地
- **图片保存目录**：设置页新增「图片保存目录」配置项（默认系统图片文件夹），与视频保存目录独立管理，方便归档
- 设置页「文件保存位置」统一改名为「视频保存目录」，与图片目录命名对称

---

## 0.4.2

### 修复 / 优化

- **videoset URL 解析**：新增从页面内 CCTV AJAX URL 字符串提取 TOPC id 的正则回退，`/videoset/` 等不暴露 `var` 声明的栏目页（如经济半小时）现可正常导入
- **栏目名多余「视频」后缀**：清洗 `<title>` 时追加去除「节目视频」「视频」「节目」后缀，同时移除冗余正则，修复 videoset 页面解析出「经济半小时视频」的问题
- **重新下载静默跳过**：设置页「↺ 重新下载」改用 `retryJob`（绕过历史去重），确保主动触发的重新下载真正入队
- **月份选择器不刷新**：`el-date-picker` 补加 `@change`，手动选月后自动刷新视频列表
- **月份「本月」按钮**：将月份快捷按钮「今」改为「本月」，语义更清晰
- **删除弹窗文案**：删除栏目确认弹窗「节目」改为「栏目」
- **红点自动消除**：点击收藏栏目后新内容红点即时消失

## 0.4.1

### 修复

- **播放按钮打开文件夹**：修复下载完成后点「▶ 播放」实际打开文件夹的问题（`applyJobFinished` 补全 `sourceJob` 赋值）
- **虚拟滚动切换不回顶**：切换栏目、月份或单视频集合时，列表不再停留在旧滚动位置

## 0.4.0

### 新增 / 优化

- **任务排序**：下载队列中排队中的任务支持拖拽（`⠿` 句柄）和「↑ 置顶」按钮调整优先级，顺序持久化到下次启动
- **历史搜索与重新下载**：设置页下载历史新增关键词搜索框；每条记录可直接点「↺ 重新下载」重新入队
- **虚拟滚动**：视频列表超过 100 条时自动切换轻量虚拟滚动，避免大量历史视频时卡顿
- **托盘失败提示**：批次下载有任务失败时，系统托盘图标标注提醒（macOS 标题 `⚠`，其他平台 tooltip），5 秒后自动复位

## 0.3.5

### 优化 / 修复

- **并行下载线程均分**：调整为每路视频实际使用 `floor(threadCount / concurrentVideos)` 个线程，总并发保持不变，避免随视频数倍增
- **设置页文案优化**：所有描述去掉技术内部术语（「分片」「CDN」「去重」），标点统一，面向用户友好表达
- **按钮 UI 重设计**：「保存设置」与「下载此集/视频」按钮改为胶囊形圆角，加柔和阴影与 hover 上浮动效
- **代码修复**：消除产品代码中唯一一处 `any` 类型；修复 `resumePending([])` 空数组时误触发 `batchFinished` 的 bug
- **测试补充**：补 `removeFromDownloadHistory`、`concurrentVideos` clamp、`resumePending` 空数组守卫、`threadCount` 均分共 5 个新测例（单元测试 → 377 个）
- **文档更新**：README/USAGE/FAQ 补充并行下载、侧边栏快捷键（`Ctrl+\`）、历史详情、线程均分行为等新功能说明
- **截图重拍**：`home.png`、`home-dark.png`、`settings.png` 对应最新 UI

## 0.1.0

cctvdl 的首个公开版本：一款开箱即用的跨平台央视视频下载器。以下是本版本提供的能力。

### 下载

- 导入栏目 / 节目链接，按月浏览、关键词搜索，封面与简介预览
- 多选批量下载，串行队列执行，任务卡片实时显示进度、速度与剩余时间
- 自动处理央视加密节目，下载后即可正常观看
- 清晰度可选：流畅 / 标清 / 高清 / 超清 / 蓝光（最高 1080p），或「自动」取最高，缺档自动降级
- 下载完成自动合并为 MP4（默认无损快速，可选兼容重编码）
- 断点续传：失败 / 取消后保留进度，重试自动续传
- 下载历史去重，重复内容自动跳过

### 管理与界面

- 下载页分「进行中 / 已完成 / 失败·取消」三组；完成后可一键播放（系统播放器）或定位文件
- 垂直侧边栏（可展开 / 收起）、深色模式、6 种主题色、封面大图预览
- 拖放导入、键盘快捷键（`1`/`2`/`3`、`F5`、`Ctrl+A`、`Delete`、`Esc`）、系统托盘进度提示、窗口大小 / 位置记忆
- 设置：保存位置、并发数、清晰度、合并方式、深色模式、主题色、日志级别 / 目录、下载历史管理

### 开箱即用

- 支持 Windows / macOS / Linux
- 安装包已内置所需运行时与 ffmpeg，无需任何额外安装或配置

### 质量

- 三层测试：单元测试 + GUI 自动化测试 + 端到端联网管线测试
- CI 在三平台逐层把关（lint / typecheck / build → 单元 → GUI）；端到端联网测试因 CDN 封境外 IP 改为手动触发
- ESLint；打 `v*` tag 时自动构建并发布三平台安装包
